# Saludproactiva
Es un sistema de gestión médica con recordatorios inteligentes e inteligencia artificial.  Su objetivo es ayudar a médicos y pacientes en el seguimiento de turnos, estudios y controles periódicos de salud.

## 🚀 Funcionalidades principales

### 🔹 Base del sistema
- Registro y login de **médicos y pacientes**.
- Gestión de **turnos** y **estudios médicos**.
- Base de datos en **MySQL** con tablas para:
  - `usuarios` (médicos/pacientes)  
  - `pacientes` (datos personales, enfermedades)  
  - `estudios` (tipo, fecha, frecuencia recomendada)  
  - `turnos` (fecha, médico, paciente, estado)  
  - `alertas` (recordatorios automáticos)
